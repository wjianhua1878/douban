# douban
仿豆瓣评分微信小程序 不定期会更新其他功能
> 使用的接口为豆瓣提供的api，所有接口地址可以在`service/api/index.js`中查看

## 实现的功能
+ 即将上映电影、热映电影、电视剧、综艺等列表及查看更多功能
+ 搜索功能
+ 电影详情页面(评分、评论、电影信息等)
